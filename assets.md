## Images Required (To Be Provided by You)

### 1. Hero Background Image
- **Purpose:** Full-width background for the hero section
- **Suggested Content:** Clean photo of the shop interior (shelves with electrical products) or shop exterior with signboard
- **File Name:** `hero-bg.jpg`
- **Dimensions:** 1920 x 900px (or higher)
- **Path after placement:** `image/hero-bg.jpg`

### 2. Shop Image
- **Purpose:** About section - shows your real shop to build trust
- **Suggested Content:** Front view of TYM Electricals & Hardware shop
- **File Name:** `shop-front.jpg`
- **Dimensions:** 800 x 600px (minimum)
- **Path after placement:** `image/shop-front.jpg`

### 3. Product Category Images (6 Images)

| Category | Suggested Content | File Name |
|----------|-------------------|-----------|
| Electrical Wires & Cables | Coils of wire / cable drums | `cat-wires.jpg` |
| Switches & Switchgear | Modular switches, MCBs on display | `cat-switches.jpg` |
| PVC Pipes & Fittings | Stacked pipes and fittings | `cat-pipes.jpg` |
| Hardware & Tools | Pliers, screwdrivers, tool sets | `cat-tools.jpg` |

- **Dimensions:** 600 x 500px (minimum) per image
- **Path:** All inside `image/` folder

### 4. Placeholder Image (Fallback)
- **Purpose:** Used if a category image is not yet provided
- **File Name:** `placeholder.jpg`
- **Content:** Simple grey or white box with camera icon
- **Path:** `image/placeholder.jpg`

## Notes
- All images should be **JPG or PNG** format
- Keep file sizes optimized (under 200KB per image preferred)
- Avoid images with watermarks
- If you do not provide an image for a section, the website will use a solid color background or placeholder
- Once you place the images in the `image/` folder, update the filenames in `index.html` if needed
